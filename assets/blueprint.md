## 6. Размещение триады Config / Client / Storage (Step 6)

| Триада | Уровень | Внешний сервис | Обоснование |
|--------|---------|----------------|-------------|
| Config | module / stage_X / нет | — | <почему здесь> |
| Client | module / stage_X / нет | <API/сервис> | <почему здесь> |
| Storage | module / stage_X / нет | <БД/файл/кэш> | <почему здесь> |

- [ ] Ни одна настройка не дублируется между уровнями (один владелец)
- [ ] Триада не привязана к операции (иначе развернуть операцию в этап)
- [ ] Инициализация: Config → fail-fast валидация → инжекция в Client/Storage
